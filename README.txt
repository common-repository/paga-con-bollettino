Gateway to pay with Bollettino Postale

== Description ==

You can add another payment method on your shop Woocommerce! With Bollettino Postale client can pay you in very simple and easy way. From internet banking, sisal and more.

= How to use it =

After installing and activating the plugin, client can see Paga con Bollettino in Checkout pay and follow the instruction.


== Installation ==

This section describes how to install the plugin and get it working.

1. Download and install the Plugin
1. Upload the plugin folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go in 'Woocommerce'/ Setting/Checkout and configure it.
